Car Inventory RESTful API Documentation

Welcome to the Car Inventory API documentation. This API allows you to manage a car inventory, including creating, reading, updating, and deleting car records. Below you'll find the necessary information to effectively utilize the API, including URL design, sample examples, and input/output specifications.

Base URL

All endpoints are relative to the base URL:

http://localhost:2000

Endpoints

Web Service 1: Inventory

Get All Cars

Retrieve a list of all cars in the inventory.

Request
GET /cars
Response
{
    "status": "success",
    "data": [
        {
            "id": 1,
            "manufacturer": "Toyota",
            "model": "Camry",
            "year": 2020,
            "color": "Red",
            "price": 25000,
            "mileage": 15000
        },
        ...
    ]
}

Get Car by ID

Retrieve details of a specific car by its ID.

Request
GET /cars/{id}
Response
{
    "status": "success",
    "data": {
        "id": 1,
        "manufacturer": "Toyota",
        "model": "Camry",
        "year": 2020,
        "color": "Red",
        "price": 25000,
        "mileage": 15000
    }
}

Create a New Car

Add a new car to the inventory.

Request
POST /cars
Body
{
    "manufacturer": "Toyota",
    "model": "Camry",
    "year": 2020,
    "color": "Red",
    "price": 25000,
    "mileage": 15000
}
Response
{
    "status": "success",
    "message": "Car added successfully",
    "data": {
        "id": 1,
        "manufacturer": "Toyota",
        "model": "Camry",
        "year": 2020,
        "color": "Red",
        "price": 25000,
        "mileage": 15000
    }
}

Update a Car

Update the details of an existing car by its ID.

Request
PUT /cars/{id}
Body
{
    "manufacturer": "Toyota",
    "model": "Camry",
    "year": 2020,
    "color": "Blue",
    "price": 26000,
    "mileage": 14000
}
Response
{
    "status": "success",
    "message": "Car updated successfully",
    "data": {
        "id": 1,
        "manufacturer": "Toyota",
        "model": "Camry",
        "year": 2020,
        "color": "Blue",
        "price": 26000,
        "mileage": 14000
    }
}

Delete a Car

Delete a car from the inventory by its ID.

Request
DELETE /cars/{id}
Response
{
    "status": "success",
    "message": "Car deleted successfully"
}

Web Service 2: Workshop

Move Car to Workshop

Move a car to the workshop by its ID.

Request
PUT /cars/{id}/workshop
Response
{
    "status": "success",
    "message": "Car moved to workshop successfully"
}

Complete Modification

Mark a car modification as complete by its ID.

Request
PUT /modifications/{id}/complete
Response
{
    "status": "success",
    "message": "Modification completed successfully"
}
